# Spotlight

Spotlight 是一個使用 [Tailwind CSS](https://tailwindcss.com) 和 [Next.js](https://nextjs.org) 建構的 [Tailwind Plus](https://tailwindcss.com/plus) 網站模板。

## 開始使用

要開始使用此模板，請先安裝 npm 依賴套件：

```bash
npm install
```

接下來，在專案根目錄建立一個 `.env.local` 檔案，並將 `NEXT_PUBLIC_SITE_URL` 變數設定為您網站的公開 URL：

```
NEXT_PUBLIC_SITE_URL=https://example.com
```

然後，執行開發伺服器：

```bash
npm run dev
```

最後，在瀏覽器中開啟 [http://localhost:3000](http://localhost:3000) 即可查看網站。

## 自訂設定

您可以透過修改 `/src` 資料夾中的檔案來編輯此模板。當您編輯這些檔案時，網站會自動更新。
